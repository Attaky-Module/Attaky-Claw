ESP-Claw Skills Lab is a community platform for sharing Attaky Claw Skills. You can find a variety of Attaky Claw Skills and Lua scripts there, and share your own Skills and Lua scripts with others.

## Install a Skill or Lua Script from Skills Lab on Your Device

Attaky Claw includes a built-in Skill for installing Skills from Skills Lab. Follow the instructions on Skills Lab and send the prompt to Attaky Claw, then Attaky Claw can automatically install the Skill or Lua script from Skills Lab.

<Aside type="caution" title="Security Reminder">

The Skills and Lua scripts on Skills Lab are provided by third parties. They may contain malicious code that could damage the device filesystem or leak sensitive credentials. We recommend carefully reviewing the contents of a Skill or Lua script before installing it.

</Aside>

## Contribute Skills or Lua Scripts to Skills Lab

You can help more people benefit from Attaky Claw by contributing Skills or Lua scripts to the [ESP-Claw Skills Lab GitHub repository](https://github.com/espressif/esp-claw-skills-lab).

Projects on ESP-Claw Skills Lab must be packaged as a Skill that follows the [ESP-Claw Skill specification](https://github.com/espressif/esp-claw/blob/master/components/common/skill_builder/claw-skill-spec.md).
This is because a standalone Lua script cannot provide enough information to the LLM. The LLM would not know what the script does or what parameters should be passed to it.

Please place your Skills, Lua scripts, and resource files under the Skills directory of the [ESP-Claw Skills Lab GitHub repository](https://github.com/espressif/esp-claw-skills-lab).
When contributing a Skill, you should ensure you have the legal right to submit it for redistribution under the MIT License—
for example, because you created the work, you have authorization from the copyright holder, or third-party material is used under a compatible license (such as CC0) that permits redistribution under MIT.
